# ✨ About Project

**Mandala Website** adalah platform berbasis web yang dirancang untuk membantu proses pengolahan, manajemen, dan visualisasi data secara modern dan efisien.

Website ini dikembangkan dengan fokus pada:

- ⚡ Kecepatan dan performa
- 🎨 Tampilan modern dan responsif
- 📊 Pengolahan data otomatis
- 📁 Export & import file
- 🔍 Monitoring dan visualisasi data
- 🧩 User Experience yang sederhana dan mudah digunakan

---

# 🛠️ Features

## 📊 Data Processing
- Rekap data otomatis

- Grafik diagram batang

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/hyugenk/mandala.git
```

## Masuk ke Folder

```bash
cd mandala
```

## Jalankan Project

```bash
npm install
npm run dev
```

atau jika menggunakan PHP:

```bash
php artisan serve
```
