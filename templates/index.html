MANDALA – Manajemen Data Izin Tinggal & Status Keimigrasian is a web-based platform developed to support the management, filtering, and processing of immigration data in the Residence Permit and Immigration Status Section at Surakarta Immigration Office. This website is designed to improve administrative efficiency and data organization through integrated features such as Application by Domicile, Children with Dual Citizenship Data Recap, and Guarantor Data Merger.

APPLICATION BY DOMICILE

Applications grouping according to working area, which as well includes final calculation of applicants' gender

CHILDREN WITH DUAL CITIZENSHIP DATA RECAP

A comprehensive data recap of dual-citizen children, including complete information to ensure the data is well-structured and easily processed for long term use

GUARANTOR DATA MERGER

An advanced feature to merge Dual Citizenship data recap with the guarantors' phone numbers, allowing both datasets to be fully integrated
